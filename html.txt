	 2. meta 主要用于设置网页中的一些元数据，元数据不是给用户看的
	 			charset 指定网页的字符集
				name   指定的数据名称
				content 指定的数据的内容
				keyword 表示网页的关键字，打上这个后当你在搜索引擎输入‘关键字’会出现你的网页
						可以同时指定多个关键字，用逗号隔开
				description 用于指定网站的描述
				refresh 将页面重定向到其他页面
				例子：<meta name="868688" content="Html">
				      <meta keyword="YOU" content="KU">
					  <meta http-equiv="refresh" content="3;url=https://www.baidu.com"> 表示3秒后跳转到百度 
	
	  在页面中独占一行的元素称为块元素，块元素用来布局
	  在页面中不会独占一行的元素称为行内元素，行内元素用来包裹文字
	  块元素中放行内元素
	  header 标签：表示网页的头部
	  aside  标签：表示网页的侧栏
	  nav    标签：表示网页中的导航

	  em     标签：用来表示语音语调的一个加重

  blockquote 标签: 用来引用一个长引用
      q      标签: 表示一个短引用

			无序：

				</ol>

				<dl>

						<dd></dd> 用来对内容进行解释说明

	4.超链接（行内元素）
	    1.使用a标签来定义超链接，a标签中可以嵌套任何的元素
		    <a href="http://www.url.com" target="_blank">Anchor Text</a>
		     	属性：
				   1.href : 指定跳转的目标路径，也可以是内部的地址
					     设置为# 可以直接跳转到页面的底部
				   2.target: 
						1) _self 默认值 在当前页面中打开超链接
						2) _blank 在一个新的页面打开超链接
				   3.id   : 每个元素都可以添加一个id属性
						 id属性是元素的唯一标识，同一个页面不能出现重复的id标识
						    例子：
							<a id="bottom" href="#">回到顶部</a>
							<a href="#bottom">回到底部</a>
	   2.图片标签
	   	使用img来引入便签（替换元素，块和行内元素之间）
			<img src="" alt=""/>
				src：表示引入图片的路径，可以是内部的，也可以是外部的
				alt：表示对图片进行描述，搜索引擎会根据alt的内容来识别图片
		图片格式
			1) jpeg(jpg)
				--支持的颜色比较丰富，不支持透明效果，不支持动图
				--一般用来显示图片
			2) png
				--支持的颜色丰富，支持透明效果，不支持动图
			3)gif
				--支持的颜色比较少，支持简单透明，支持动图
				--
			4)webp
				--它具备其他图片格式的所有优点，并且文件特别小
 			5)base64 
				--将图片使用base64编码，可以将图片变成字符，通过字符引入图片
				--当你希望加载网页是也加载图片可以使用这个
		引入音频

			  属性：
			  	1）controls ：是允许用户控制播放
				2）loop     ：用于循环播放
				3）autoplay ：是否自动播放
			采用以下方式更加:
			使用source

			2. <video src="" height="" width="" preload="none" autoplay="autoplay"></video>
